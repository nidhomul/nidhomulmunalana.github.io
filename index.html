Belajar Grid System

Apa sih Grid itu? Penting tidak Grid di terapkan pada website?

Menggunakan Grid System memudahkan kita dalam mengatur tata letak elemen saat membuat website. Zaman dahulu, kebanyakan website menggunakan tag table untuk mengatur tata letak (layout-nya). Sekarang sudah tag table tidak digunakan lagi, karena kurang efektif dan tidak SEO friendly. Oleh karena itu, website zaman sekarang menggunakan CSS untuk mengatur tata letaknya. Sistem Grid di Bootstrap mengadopsi konsep tabel. Karena itu, Kita hanya perlu menggunakan tiga kelas untuk membuatnya. Kelas tersebut kita terapkan dalam tag 'div'.
Berikut ini ketiga kelas yang dimaksud, diantaranya:
1. Kelas .container: Kelas kontainer berfungsi membungkus konten website. Kelas ini sama fungsinya seperti tag 'table'.
2. Kelas .row: Berfungsi untuk membuat baris (penempatanya di dalam Kelas .container),
3. Kelas .col: berfungsi untuk membuat kolom (penempatanya di dalam Kelas .row).

Grid System yang ada di Bootstrap mempunyai fungsi sendiri-sendiri, diantaranya:

  • col-lg-*: digunakan untuk mengatur grid pada monitor komputer yang berukuran besar.
  • col-md-*: digunakan untuk mengatur grid pada layar monitor komputer yang berukuran sedang.
  • col-sm-*: digunakan untuk mengatur grid pada monitor yang berukuran tablet.
  • col-xs-*: digunakan untuk mengatur grid untuk ukuran handphone.
